## 소개

담요는 지도 상에서 내 근처에 있는 흡연구역들을 알려주는 서비스 입니다.<br>

흡연을 하기 위해 흡연구역을 찾아다니신 적이 있으신가요?<br>
애매한 구역에서 흡연을 하며 주변 눈치를 보신 적이 있으신가요?<br>
흡연을 마친 뒤 담배꽁초를 길바닥에 버리고 계신가요?<br>

그렇다면 담요와 함께 올바른 흡연 문화를 만들어요!
